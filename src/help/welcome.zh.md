## Markdown,日常会用到的部分

Levis 像 Typora 一样所见即所得地渲染 Markdown——没有预览面板,你看到的就是最终的文档。

- 输入 `#` 加一个空格开始一个标题。可以在下面的空行试试。
- 文字两边加 `**` 变成**粗体**,加 `*` 变成*斜体*——输入标记符号,看它们即时变成真正的格式。
- 以 `-` 或 `1.` 开头的一行会自动变成列表项。

## AI 续写——不用任何配置

把光标放在这句话的最末尾,停顿一下,看看接下来会

按 <kbd>Tab</kbd> 采纳建议,或者继续打字忽略它。任何文档的第一句话就能触发,没有最小字数限制。如果没有出现建议,检查一下是否已经登录了某个 AI provider(**设置 > AI**),以及那里的续写功能是否开启。

## 询问 AI——选中点什么试试

选中这整段文字(你可以让它重写、扩写、缩写,或者改得更正式——什么要求都行),然后按 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>K</kbd>。一个小对话框会在选中处打开,选中的内容已经自动带上作为上下文。

让它重写上面这段。如果你的 provider 支持(ChatGPT、OpenAI API Key 或 Claude),修改会直接出现在文档里——删除的文字加删除线,新增的文字以绿色幽灵文本显示,旁边浮现一个小面板,**✓ 采纳** / **✗ 拒绝**。采纳前文档不会有任何变化,<kbd>Cmd</kbd>+<kbd>Z</kbd> 可以一步撤销已采纳的修改。

对话弹窗会记住过去的对话——头部的时钟图标可以展开历史记录下拉菜单,侧栏的**对话**标签页显示同一份列表。

## 还有几个值得知道的功能

- **语法检查**会在你写作时给可能有问题的地方加下划线;悬停查看修改建议,或用 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>G</kbd> 立即检查当前段落。
- **侧栏**(用 <kbd>Cmd</kbd>+<kbd>\\</kbd> 切换)有文件树、文档大纲、剪贴板历史和对话历史几个标签页。
- **源码模式**(<kbd>Cmd</kbd>+<kbd>/</kbd>)可以随时切换到查看原始 Markdown 文本。
- 这里出现的每一个快捷键都可以在**设置 > 快捷键**里重新绑定。
