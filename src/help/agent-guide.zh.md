# Levis AI Agent 指南

Levis 内置一个写作专用的 AI Agent。这份文档介绍如何登录、三个 AI 功能怎么用，以及如何用「Agent 工作区」把它定制成你自己的写作助手。

> [!TIP]
> 所有 AI 功能都可以在 **设置 > AI** 里单独开关。

## 登录 / 配置提供方

在 **设置 > AI > 提供方** 里选择一种方式：

| 提供方 | 说明 |
| --- | --- |
| **ChatGPT (Codex)** | 用你的 ChatGPT 账号登录（浏览器授权，无需 API Key）。功能最全：只有它支持工具调用——修改提议、读文件、技能动态加载、联网搜索。 |
| **Claude** | 用你的 Claude 账号登录（浏览器授权）。 |
| **OpenAI API Key** | 粘贴一个标准的 OpenAI API Key。 |
| **自定义接口** | 任何 OpenAI 兼容的接口（如本地模型、中转服务）：填接口地址、可选的 Key，然后获取或手填模型名。 |

点击「使用 ChatGPT 登录」/「使用 Claude 登录」后会打开浏览器完成授权，回到应用即完成。

## 三个 AI 功能

### 1. 续写（补全）

输入时自动在光标处给出灰色的续写建议，按 <kbd>Tab</kbd> 采纳。也可以用 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>Space</kbd> 或右键菜单手动触发。

在 **设置 > AI** 里可以调整**续写语气**（正式 / 轻松 / 学术 / 简洁），还可以写一段**续写指令**（例如「不要使用破折号」），会附加到每次补全请求。

### 2. 语法检查

对可能有问题的文字加下划线，悬停查看说明并一键应用修改。快捷键 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>G</kbd> 立即检查当前段落。

### 3. 询问 AI（对话 Agent）

按 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>K</kbd>（或右键 > 询问 AI）在光标处打开对话框。它了解整篇文档；如果打开前选中了文字，选中内容会自动作为上下文。

- **修改提议**：让它改文档时（ChatGPT 提供方），它会给出带增删对照的修改卡片，点「应用」才真正写入文档，<kbd>Cmd</kbd>+<kbd>Z</kbd> 可撤销。
- **附加文件**：点输入框左侧的 **+** 可以把任意文本文件附到这条消息里（例如资料、大纲、另一章）。
- **联网搜索**：在设置里开启后（仅 ChatGPT 提供方），Agent 需要时可以自己搜索网页。
- **读工作区文件**：文档已保存时（仅 ChatGPT 提供方），Agent 可以自己列出并阅读文档所在文件夹里的文件——把参考资料放在文档旁边即可。

## Agent 工作区

不同的写作项目需要不同的指令、技能和资料。工作区就是**文档所在的文件夹**，在里面放一个 `.levis/` 目录即可定制 Agent：

```
我的小说/
├── 第一章.md          ← 正在编辑的文档
├── .levis/
│   ├── agent.md       ← 常驻指令：文风、术语表、项目背景
│   └── skills/
│       ├── polish.md  ← 技能：一个文件一个技能
│       └── outline.md
└── 资料/
    └── 设定.md         ← Agent 可以自己读取的参考资料
```

另外还有一个**全局层**，对所有文档生效（设置 > AI > Agent 工作区 > 打开全局文件夹），结构相同。工作区里同名的技能会覆盖全局的。

### agent.md — 常驻指令

`agent.md` 里的内容每次对话都会告诉 Agent。适合放：写作风格要求、人名地名术语表、这个项目是什么。

### skills/ — 技能

一个技能 = 一个 Markdown 文件：开头是 YAML 说明，正文是完整指令。

```markdown
---
name: polish
description: 润色选中的段落，保持原意，收紧句子
---

对给出的文字逐句润色：删掉冗余的词，把长句拆短，
保持原本的意思和语气不变。只输出润色后的文字。
```

技能有两种触发方式：

1. **手动**：在对话框输入 `/`，从菜单里选（或直接输 `/polish 再帮我看看这段`）。
2. **自动**（仅 ChatGPT 提供方）：Agent 知道每个技能的名字和描述，当你的请求和某个技能的描述匹配时，它会自己加载并遵循完整指令。

> [!NOTE]
> 技能文件改完即生效——下次打开对话框就会用新的内容，无需重启。

## 隐私说明

文档内容、附加的文件和工作区指令会随请求发送给你所选的提供方。联网搜索由 OpenAI 在服务端执行。不需要时在设置里关掉对应功能即可。
