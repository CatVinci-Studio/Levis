## Markdown, the parts you'll use daily

Levis renders Markdown as you type, Typora-style — no preview pane, what you see is the document.

- Type `#` followed by a space to start a heading. Try it on the blank line below.
- Wrap text in `**` on both sides for **bold**, or `*` for *italic* — type the markers and watch them turn into real formatting.
- A line starting with `-` or `1.` becomes a list item automatically.

## AI completion — no setup needed

Put your cursor at the very end of this sentence, pause for a moment, and watch what

Press <kbd>Tab</kbd> to accept it, or just keep typing to ignore it. It works from the very first sentence of any document — there's no minimum length. If nothing appeared, check that you're signed in to an AI provider (**Settings > AI**) and that completion is enabled there.

## Ask AI — select something and try it

Select this whole paragraph (you can rewrite it, expand it, summarize it, or make it more formal — literally anything) and press <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>K</kbd>. A small chat box opens right at your selection with the text already attached as context.

Ask it to rewrite the paragraph above. If your provider supports it (ChatGPT, OpenAI API Key, or Claude), the change appears directly in the document — struck-through text for what's removed, green ghost text for what's added — with a small **✓ Accept** / **✗ Reject** panel next to it. Nothing is written until you accept, and <kbd>Cmd</kbd>+<kbd>Z</kbd> undoes an accepted edit in one step.

The chat popup remembers past conversations — the clock icon in its header opens a history dropdown, and the sidebar's **Chats** tab shows the same list.

## A few more things worth knowing

- **Grammar check** underlines possible issues as you write; hover one to see the fix, or trigger it on the current paragraph with <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>G</kbd>.
- The **sidebar** (toggle with <kbd>Cmd</kbd>+<kbd>\\</kbd>) has tabs for your file tree, the document outline, clipboard history, and chat history.
- **Source mode** (<kbd>Cmd</kbd>+<kbd>/</kbd>) shows the raw Markdown text if you ever want it.
- Every shortcut here is rebindable in **Settings > Shortcuts**.
